# dinudsi.github.io

Personal portfolio site for **Dinudsi** — Full Stack Architect, AI Engineer & Product Strategist.

## Live Site

[https://dinudsi.github.io](https://dinudsi.github.io)

## Features

- Modern dark theme with glassmorphism effects
- Interactive particle background with mouse tracking
- Typing animation cycling through roles
- Smooth scroll-triggered fade-in animations
- Animated stat counters
- 3D tilt effect on project cards
- Fully responsive (mobile, tablet, desktop)
- Zero dependencies — pure HTML, CSS, and vanilla JS

## Sections

- **Hero** — Introduction with animated code window
- **About** — Background, values, and quick info
- **Tech Stack** — Categorized technology expertise
- **Projects** — Featured work showcase
- **Architecture** — Engineering highlights and philosophy
- **Contact** — Email and GitHub links

## Deployment

Push to the `main` branch and enable GitHub Pages in repository settings (source: root of `main` branch).
